## 第一章 简单操作

> + 单元格显示`#`,意味着`单元格太窄`，没法全部显示

+ 自动填充
  + 操作
    + 选中单元格(需要操作的行或者列)
    + 点击`开始`
    + 点击`编辑`
    + 点击`填充`
    + 点击`序列`
  + `递增填充`
    + 选中单元格，放到右下角，按住`ctrl`,往下选择单元格
    + 选中单元格，放到右下角,往下选择单元格
  + `类型`
    + **等差序列**
    + **等比序列**
    + **日期序列**
