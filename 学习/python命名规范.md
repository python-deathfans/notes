## 常见命名方式

+ b
  + 小写字母
+ B
  + 大写字母
+ lowercase
  + 小写字母
+ **lower_case_with_underscores**
  + 使用下划线分割的小写字母
+ UPPER CASE
  + 大写字母
+ UPPER_CASE_WITH_UNDERSCORES
  + 使用下划线分隔的大写字母
+ **CapitalizedWords**
  + 驼峰命名
+ **mixedCase**
  + 不同于首字母大写，第一个单词的首字母小写

+ `包名和模块名`
  + **模块**应该用简短全小写的名字，如果为了提升可读性，**下划线也是可以用的**。Python**包名**也应该使用简短全小写的名字，但**不建议用下划线**。
+ `类名`
  + **首字母大写**
+ `函数名`
  + **全小写，可以用下划线分割**
